Features

Create Books (title, description, username)

Create Chapters under selected Books

Create Pages under selected Chapters

Maintain Pages and Versions as separate arrays

Restore Pages to previous states

Automatically create a new version on restore

Remove newer pages when restoring an older page

Simple JSON-based data storage

RESTful API structure


Tech Stack

-Node.js

-Express.js

-LowDB / JSON File Storage

-NanoID (for unique IDs)

-JavaScript
